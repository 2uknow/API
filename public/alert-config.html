웹훅 URL 설정

알람을 받으려면 config/settings.json 파일에 웹훅 URL을 설정해야 합니다:

{
  "webhook_url": "https://your-webhook-url.com/webhook",
  "run_event_alert": true,
  "alert_on_start": true,
  "alert_on_success": true,
  "alert_on_error": true,
  "alert_method": "flex"
}

또는 환경변수 NW_HOOK을 설정할 수도 있습니다.
