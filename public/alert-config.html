🔗 네이버웍스 웹훅 URL 생성하기

  1. 네이버웍스 관리자 센터에 로그인
  2. Bot > 외부 연동 Bot 메뉴 이동
  3. "Bot 추가" 버튼 클릭하여 새 Bot 생성
  4. 생성된 Bot의 "Webhook URL" 복사
  5. 위 입력란에 URL 붙여넣기

⚙️ 알람 타입별 설명

  • 시작 알람: API 모니터링 작업이 시작될 때 전송
  • 성공 알람: 모든 테스트가 성공적으로 완료될 때 전송
  • 실패 알람: 테스트 실패 또는 오류 발생 시 전송

🎨 메시지 형식 차이점

  • Flex 메시지: 카드 형태의 풍부한 UI, 버튼 및 링크 포함
  • 텍스트 메시지: 간단한 텍스트 형태, 빠른 알림용
